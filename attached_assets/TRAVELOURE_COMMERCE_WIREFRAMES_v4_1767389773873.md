## CORE PHILOSOPHY

### The New Approach: Show, Don't Ask

**TRADITIONAL APPROACH:**
```
Ask Budget â†’ Ask Preferences â†’ Show Curated Results â†’ Hope They Book
Problem: Friction before value, no price transparency, no AI monetization
```

**TRAVELOURE APPROACH:**
```
Minimal Input (Location + Dates) â†’ Browse Real Inventory â†’ Add to Cart â†’ 
See Total Cost â†’ Offer AI Optimization â†’ Compare Savings â†’ Choose Plan
Benefit: Immediate value, transparent pricing, monetizable AI upgrade
```

### Key Principles

1. **Ask Almost Nothing**: Just location and dates to start
2. **Show Real Inventory**: Actual bookable content from affiliates (GetYourGuide, Viator, Klook, etc.)
3. **Let Users Build Cart**: Like shopping on Amazon - browse, add, total updates
4. **Profile Implicitly**: Learn from cart selections (budget tier, interests, style)
5. **Prove AI Value**: Generate 2-3 optimized alternatives showing concrete savings
6. **Monetize Optimization**: Charge for AI-enhanced plans that demonstrably save money/time

3. **Control** - Users build their own plan
4. **Proven value** - AI shows "You could save $327" with real numbers
5. **Low-risk upgrade** - $20-50 for AI vs potential $200-400 savings
